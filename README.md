# <img src="https://polybox.ethz.ch/index.php/s/c8uZqrzwk45wpBx/download?path=%2Fpng&files=zonda-high-resolution-logo-transparent.png" width="110" valign="middle" alt="zonda"/> Request


Backend service for generating external parameter data on ICON grids.

This repository contains the processing logic behind Zonda Request handling. 
Requests are triggered by creating a [data request issue](https://github.com/C2SM/zonda-request/issues/new?template=data_request.md). The configurations for these requests are usually generated via the [Zonda Web Interface](https://zonda.ethz.ch/).

## Documentation

Comprehensive documentation for the Zonda Request backend is available at:

https://zonda.ethz.ch/docs/backend
